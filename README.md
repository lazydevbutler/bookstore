# To run the project

## Pre-requisite

- Yarn/Npm
- Nodejs version 12 and above
- Java 8 and above
- Mysql

## React

to install the node_modules dependencies

```
yarn install
```

to run the react application

```
yarn dev
```

## Springboot

```
mvn clean
```

```
mvn package
```

```
java -jar target/demo-0.0.1-SNAPSHOT.jar
```

## Mysql

Db scripts are found in `db` folder
